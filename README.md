## Getting Started

### Follow these steps to set up and run the development server for your Next.js project.

#### First, download all dependencies using your preferred package manager:

```bash
npm i
# or
yarn i
# or
pnpm i
# or
bun i
```

#### Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
